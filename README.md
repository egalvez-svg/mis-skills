# Agent Skills Repository

Este repositorio contiene skills para agentes de IA compatibles con el CLI `vercel-labs/skills`.

## Skills Disponibles

### [yii2-to-prisma](./skills/yii2-to-prisma)
**Autor: Eduardo Galvez**

Transforma consultas de Yii2 Active Record y Query Builder a sintaxis de Prisma Client.

## Cómo Instalar

Para instalar este skill en tu proyecto o editor, usa el CLI de `skills`:

```bash
npx skills add egalvez-svg/mis-skills --skill yii2-to-prisma
```

Sigue las instrucciones en pantalla para seleccionar tu agente (Antigravity, Cursor, Cline, etc.).
